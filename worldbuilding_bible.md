# 世界观圣经 v0.1

## 一句话概念
在一片被虚空潮汐撕裂的前线废土上，玩家作为 Hub 的组织者，带领三名各自背负旧誓与旧伤的兽人战士反复进入危险路线，回收资源、修复构筑、选择撤离或继续深入，并在一次次归来中把失控的概率重新变成可以托付的未来。

## 核心理念
本项目的核心不是“随机带来刺激”，而是“人在随机中建立信任”。

骰子代表世界的不稳定性，也代表角色自身的战斗节奏。玩家不能完全消除风险，只能通过构筑、装备、撤离判断和关系推进，把风险变成可承受、可利用、可记忆的选择。

三层体验对应三层主题：
- 战斗层：面对不可控结果，仍然做出清醒排序。
- Run 层：在贪取收益和保住明天之间做判断。
- 情感层：理解角色为何冒险，并让他们愿意把更危险、更真实的自己交给玩家指挥。

## 世界现状
世界曾由多个前线航道、边境城市和荒野聚落连接。后来虚空潮汐出现，空间被撕裂成一段段不稳定路线。旧设施、舰体残骸、锻炉前哨和被污染的生态区漂在裂隙边缘，形成可以短暂进入的 Run 区域。

虚空潮汐不是单纯的黑暗力量。它更像一种会扭曲概率、记忆和机械指令的灾变。被卷入其中的设施会重复旧命令，被污染的生物会保留本能却失去边界，曾经的战场会像骰子一样反复掷出不同结果。

因此，前线生存依赖三件事：
- 可复现的战斗协议。
- 能带回资源和装备的撤离路线。
- 一个能让人回来的 Hub。

## Hub：前线整备点
Hub 的暂定叙事名为“前线整备站”。它不是安稳城市，而是一座拼接出来的移动据点，核心由半毁前哨、舰船维修舱、旧锻炉和信标塔组成。

Hub 的作用：
- 为出战角色校准 D6。
- 储存和修理带回的装备。
- 记录 Run 结果、撤离损失和区域变化。
- 让角色在战斗之外恢复、争执、靠近或沉默。
- 让玩家主角做出谁出战、带什么、是否继续投入某段关系的决定。

Hub 的情绪功能比物理功能更重要。玩家每次回到 Hub，都应该感觉这里仍然破旧、紧张、资源不足，但有人会因为你回来了而改变一句话、一个动作或一个决定。

## 玩家主角
玩家主角是 Hub 的组织者和战术协调者，不是常规战斗单位。

推荐身份：前线信标的继任调度官。上一任调度官在一次深层 Run 中失踪，只留下可运转但不断失真的航路记录。玩家主角继承了 Hub 的权限，也继承了一个尴尬位置：所有人都需要你的指挥，但没有人天然相信你。

玩家主角的叙事功能：
- 选择谁出战，承担派遣责任。
- 在 Hub 中承接角色的脆弱、暧昧、愤怒和信任。
- 将 Run 的失败、撤离和完成解释成下一步计划。
- 通过关系推进解锁角色更深层的战斗许可。

主角不参战的理由：
- Hub 的信标系统需要主角留在后方校准撤离窗口。
- 出战角色的 D6 构筑需要主角远程读取和排序。
- 主角亲自上场会让 Hub 失去唯一稳定锚点。

## D6 的叙事解释
D6 是“战斗校准骰”，由旧世界的概率稳定技术改装而来。每颗骰子把角色的战斗习惯、装备接口和风险代价压缩成六个可读面。

骰子不是魔法道具，也不是纯机械模块，而是角色和 Hub 共同校准出的战斗协议：
- 普通面代表可靠动作。
- 资源面代表角色进入自身节奏。
- 负面面代表身体、装备或心理代价。
- 附魔代表临时写入的战斗偏置。
- 替换和升级代表角色愿意改变自己的作战方式。

这解释了为什么构筑不是单纯数值成长。每次改骰，都是改变角色如何面对危险。

## Run 的叙事解释
Run 是从 Hub 出发进入不稳定路线的行动。每条路线只在短时间内可读，越深入，虚空潮汐越强，收益越高，撤离越难。

节点类型叙事：
- Battle：遭遇被污染的守卫、生物或旧系统。
- Event：发现信号、缓存、锻炉或幸存痕迹。
- Supply：安全补给点或可短暂停留的维修夹层。
- Shop：游离商队、自动交易机或旧军需终端。
- Rest：虚空暂退的静默口袋。
- Elite：守住稀有资源或航路钥匙的强敌。
- Boss：一条路线的稳定核心，也是对当前构筑的最终检验。

## 撤离的叙事解释
撤离不是认输，而是 Hub 的核心伦理。

旧前线曾经失败，是因为指挥系统只要求推进，不允许回头。玩家主角的 Hub 与旧系统不同：它承认“带着伤和资源回来”本身就是胜利。

撤离应传达的感觉：
- 这次没有走到终点，但有人活着回来了。
- 带回的装备和经验会改变下一次。
- 角色可能不甘心，但不会把撤离简单视为懦弱。
- 关系线可以通过撤离推进，因为撤离也是一种彼此负责。

## 装备与损坏
装备是从 Run 中带回的旧世界残片。它们不是永久安全的战利品，而是有使用痕迹和带回次数的前线记忆。

损坏叙事：
- 第一次带回代表完成登记和基础修复。
- 多次带回后，虚空残留、过载、血污和临时改造会累积。
- broken 装备不是“消失”，而是失去战斗可靠性，需要修理或分解。

装备跨角色通用的叙事理由：
- Hub 统一改装接口。
- 角色不被职业锁死，适配来自标签和作战习惯。
- 一件装备在不同角色身上会呈现不同意义，例如 Blood Whetstone 对 Cyan 是危险散热，对 Aurian 是主动承压。

## 敌人和虚空潮汐
敌人不是随机怪物，而是前线问题的具象化。

### Hub Edge Vanguard
旧前线防御系统的残留单位。它仍在执行“守住边缘”的命令，但已经无法分辨 Hub、幸存者和虚空污染。

叙事功能：
- 教玩家读懂意图。
- 让角色第一次面对“旧秩序仍然在伤害活人”。
- 适合作为 Boss 升级版本的原型。

### Reef Stalker
来自淹没生态区的高速猎手。虚空潮汐让它学会干扰信号、追踪标记和破坏重骰节奏。

叙事功能：
- 代表自然生态被扭曲后的求生本能。
- 考验玩家是否过度依赖完美结果。
- 特别刺激 Helios 的猎人身份。

### Void Howler
深层裂隙中的回声型威胁。它不像普通生物，更像被虚空潮汐反复折叠后的痛苦信号。

叙事功能：
- 检查构筑是否完整。
- 让资源、标记、Stance 这些角色核心节奏被压迫。
- 作为后续主线 Boss 或虚空核心的声音预告。

## 三名初始可出战角色
### Cyan Ryder
身份关键词：蓝龙、舰长、高科技远程引擎、过载、控制欲、失船者。

外在印象：
Cyan 看起来最像能把一切讲清楚的人。他的装备整洁、战术语速快，喜欢用屏幕、曲线和冷却时间解释问题。但他的冷静并不等于稳定。越接近失控，他越想把所有变量抓在自己手里。

来到 Hub 的原因：
Cyan 曾是小型护航舰的舰长。虚空潮汐吞掉他的航道时，他为了保住逃生窗口，把舰体反应炉推到危险阈值。部分船员获救，舰体和航海记录却坠入前线裂隙。他追踪残余信号来到 Hub，因为 Hub 的信标能读到那艘船最后的黑匣子回波。

他留在 Hub 的真正原因：
一开始，Cyan 只把 Hub 当成临时维修站和信号中继。他需要玩家主角的调度权限，也需要 Hub 的锻炉和装备仓库。随着 Run 推进，他发现主角不是要夺走他的指挥权，而是在他过载前替他保留撤离选择。关系推进的核心，是 Cyan 从“我必须控制所有变量”转向“我可以把某些变量交给你”。

玩法映射：
- Overload 是他的反应炉和心理压力。
- 冷却、Vent、Shift 是他学习放手的动作。
- 无人机路线代表他重新接受协作，而不是把所有责任集中在自己身上。

个人故事线：
1. 初遇：他要求使用 Hub 信标，只承认这是一场资源交换。
2. 磨合：他频繁质疑撤离判断，认为没有打到最深就不算有效行动。
3. 裂痕：找到失船记录，证明他当年的选择救了人，也伤了他自己。
4. 信任：他允许主角在战斗中强制冷却或中断过载链。
5. 亲密线接口：他第一次在非任务时间关闭战术屏幕，用自己的声音而不是数据谈论恐惧。

Hub 文案方向：
- “给我三分钟，我能让这套坏掉的系统再撑一轮。”
- “撤离不是问题。问题是我们有没有承认撤离也是计划的一部分。”
- “别让我满载太久。你看见读数变红时，直接叫停。”

### Helios Windchaser
身份关键词：狮鹫、游侠、猎人、标记、箭袋、路线判断、失去队伍的向导。

外在印象：
Helios 比 Cyan 更轻快，也更会缓和气氛。他习惯观察地形、风向和别人没说出口的压力。他不喜欢被称为射手，因为他认为真正的猎人不是会射中目标，而是知道什么时候不该射。

来到 Hub 的原因：
Helios 曾为边境迁徙队做向导。一次撤离路线被虚空潮汐折断，他标记了错误的安全风道，导致队伍被迫分散。他一直在寻找失散者留下的羽标和旧罗盘信号。Hub 的路线图能读到那些断续标记，所以他来到这里。

他留在 Hub 的真正原因：
Helios 最初不愿意被固定在任何据点。他相信移动才能活下去，也害怕再次成为“带错路的人”。玩家主角给他的不是命令，而是一个能让错误被记录、复盘、修正的地方。关系推进的核心，是 Helios 从“我必须独自判断每一条路”转向“我可以让别人看见我的判断过程”。

玩法映射：
- Mark 是他对世界重新建立可读性的方式。
- Quiver 是弹药，也是节奏和信心。
- Companion/Trap 路线代表他愿意重新布置团队战术，而不是一个人把所有风险背走。

个人故事线：
1. 初遇：他主动带回一条安全路线，像是路过帮忙。
2. 磨合：他总在撤离窗口前给出建议，却不愿解释建议来自什么经验。
3. 裂痕：某个标记事件暴露他曾经带错路的创伤。
4. 信任：他让主角参与标记优先级，承认自己也会看错风。
5. 亲密线接口：他把旧罗盘交给主角保管一晚，象征愿意停留。

Hub 文案方向：
- “风向不坏，只是今天它不站在我们这边。”
- “目标身上的标记别攒太久。能收就收，活着回来的猎人才有下一箭。”
- “你来定撤离点。我负责确认路还在。”

### Aurian
身份关键词：黑龙、誓约骑士、重型战士、Stance、承压、沉默、旧誓言。

外在印象：
Aurian 是最安静的人。他不像 Cyan 那样解释，也不像 Helios 那样缓和。他会站在 Hub 火边擦拭重剑，听完所有争论，然后只说一句必要的话。许多人以为他冷漠，其实他只是太习惯把痛苦放在盔甲里面。

来到 Hub 的原因：
Aurian 曾守护一座边境誓约门。那座门的旧命令要求守卫“直到最后一人”。虚空潮汐来临后，命令变成诅咒，守卫们不断被迫重复没有意义的防线。Aurian打碎誓约门，带着“背誓者”的名声离开。他来到 Hub，是因为这里的信标火焰使用了誓约门残片，他要确认那段旧誓是否仍在伤人。

他留在 Hub 的真正原因：
Aurian 不相信轻易的组织，也不相信漂亮的作战口号。但 Hub 允许撤离，这一点动摇了他。玩家主角不是让他再次“守到死”，而是让他学会在承压之后还能回来。关系推进的核心，是 Aurian 从“我只能用沉默承担”转向“我可以说出誓言，并让你听见代价”。

玩法映射：
- Stance 是他重新站稳，而不是盲目忍耐。
- Guard/Counter 是把敌人的压力转为回应。
- Blood Price 路线代表他愿意面对自毁冲动，并让主角帮他设下边界。

个人故事线：
1. 初遇：第一次 Run 后，他才在 Hub 火边承认玩家主角的指挥。
2. 磨合：他接受命令，但不解释自己的旧誓。
3. 裂痕：某次深层路线触发誓约门残响，他几乎拒绝撤离。
4. 信任：他允许主角中断处决大回合，优先保命。
5. 亲密线接口：他把重剑放在主角能碰到的位置，象征他不再把一切都握死。

Hub 文案方向：
- “火还在。那就不是结束。”
- “命令我撤。不要等我自己承认撑不住。”
- “誓言不是锁链。至少，你是这样说的。”

## 三人关系结构
三名角色不应只是平行恋爱对象，他们在 Hub 中也要互相照出主题。

Cyan 与 Helios：
- Cyan 重视控制，Helios 重视判断。
- Cyan 会质疑 Helios 的“凭风感”，Helios 会调侃 Cyan 把恐惧做成了仪表盘。
- 二人可以围绕无人机、标记和路线预判形成轻快但有刺的互动。

Cyan 与 Aurian：
- Cyan 害怕沉默，因为沉默不可读。
- Aurian 不喜欢 Cyan 把痛苦量化成数据。
- 二人关系可以从互相误解发展为战场上的强互补：一个叫停过载，一个挡住反噬。

Helios 与 Aurian：
- Helios 看得出 Aurian 不愿求援。
- Aurian 看得出 Helios 用轻松掩盖旧错。
- 二人适合写安静的夜间 Hub 对话，一个守火，一个看风。

玩家主角与三人：
- 对 Cyan：成为他允许介入的冷却阀。
- 对 Helios：成为他愿意共享路线判断的人。
- 对 Aurian：成为他能听从撤离命令的人。

## 主线推进草案
### Act 1：信标仍亮
目标：建立 Hub、三名角色和基础 Run。

剧情重点：
- 玩家继承 Hub 调度权限。
- 三名角色分别因信号、路线和誓约残片来到 Hub。
- 第一次击退 Hub Edge Vanguard，证明 Hub 还能运作。
- Aurian 的初始关系节点在第一次 Run 后触发，表现为“火边开口”。

### Act 2：路线会说谎
目标：让撤离成为价值选择，而不是失败备选。

剧情重点：
- Reef Stalker 和事件信号揭示路线图不可靠。
- Helios 的旧罗盘线展开。
- Cyan 找到失船回波，但记录不完整。
- 玩家多次面对“继续深入可能获得真相，撤离可以保住现有成果”的选择。

### Act 3：旧誓仍在命令死人
目标：把 Aurian 的誓约门、Cyan 的失船和 Helios 的错路汇合。

剧情重点：
- Void Howler 反复播放旧前线命令。
- Hub 信标被证明使用了旧誓约门残片。
- 三人的过去都与同一场前线撤离失败有关，但他们看到的是不同碎片。
- Boss 不只是怪物，而是旧系统“不允许撤离”的残留意志。

### Act 4：允许归来
目标：完成 MVP 后续主线方向的主题收束。

剧情重点：
- 玩家不是摧毁所有虚空，而是重写 Hub 的行动准则。
- 完成 Run 的意义从“深入到底”变成“带回足够改变未来的东西”。
- 三名角色各自解锁更深 BD 或关系奖励，象征他们把更真实的风险交给玩家。

## 关系节点建议
### Cyan 节点
1. `rel_cyan_signal`: 第一次使用 Cyan 完成或撤离 Run 后，他要求查看 Hub 信标日志。
2. `rel_cyan_overload`: 触发一定次数 Overload 后，他承认自己的反应炉校准过于激进。
3. `rel_cyan_blackbox`: 完成深层路线后，找到失船黑匣子片段。
4. `rel_cyan_cooldown`: 主角解锁可让 Cyan 开局获得冷却或降低 Overload 代价的奖励。
5. `rel_cyan_trust`: Cyan 允许主角在关键回合优先执行撤离协议。

### Helios 节点
1. `rel_helios_compass`: 第一次撤离后，他称赞撤离点选得不错。
2. `rel_helios_mark`: 多次触发 Mark 或击败 Stalker 后，他谈起旧罗盘。
3. `rel_helios_lost_route`: 事件节点发现失散迁徙队的羽标。
4. `rel_helios_shared_map`: 主角解锁标记保留或 Quiver 回复相关奖励。
5. `rel_helios_stay`: Helios 把罗盘留在 Hub，表示愿意回来。

### Aurian 节点
当前已有：
- `rel_aurian_intro`: 第一次 Run 后，Aurian 在 Hub 火边开口。
- `rel_aurian_trust`: 完成路线后，Aurian 接受主角指挥。

建议扩展：
3. `rel_aurian_broken_oath`: 遭遇 Void Howler 或深层事件后，誓约门残响触发。
4. `rel_aurian_withdraw`: 选择撤离后，Aurian 承认撤退命令比死守更难服从。
5. `rel_aurian_embers`: 解锁 Stance 或防反相关专属奖励，他把旧誓改写为新誓。

## UI 与短文案方向
### Hub 标题
可选：
- Hub - 前线整备
- 前线整备站
